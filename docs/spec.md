GUI Demo 的目的

在 1000×1000 的平面上，模擬「人口群聚 → 災害撤離 → 前往避難所 → 部分人超時失敗」，並用 GUI 動畫呈現與輸出指標（成功率、平均到達時間等）。

介面請完全復刻「預期UI.png」

⸻

具體規格（交辦用）

畫面
	•	1000×1000 平面映射到視窗 Canvas（例如 700×700 px）
	•	避難所：藍色方塊
	•	人口（代理人）：移動中灰色點；到達綠色；失敗紅色
	•	可顯示「群聚中心」作為淡紫色圈（可選）

資料生成
	•	避難所：在平面內均勻隨機生成 N 個
	•	人口：用 K 個群聚中心（Gaussian Mixture）生成 M 個代理人，呈現不均勻/群聚現象

撤離規則
	•	每個代理人分配最近避難所（Euclidean）
	•	每個 tick 依速度往目標移動（直線追蹤）
	•	速度：成人與高齡（或慢速族群）不同，並加入隨機波動
	•	災害期限 T：超過則判定「來不及」→ 失敗（標紅）

互動
	•	Generate：重新生成避難所與人口
	•	Start / Pause / Reset
	•	可調參數：避難所數、人口數、群聚數、期限 T、成人/慢速比例、速度、tick 時間
	•	可選「Instant Run」：不動畫、直接跑完拿結果

輸出指標
	•	成功率、失敗率
	•	平均到達時間、P90 到達時間（成功者）
	•	視覺化結果（動畫＋最後狀態）
